-git init:  Khởi tạo git cho một project ( chỉ cần dùng 1 lần duy nhất)

-git add:Sắp xếp cái file đã bị thay đổi, thêm mới or xóa so với lần commit trước đó, để chuẩn bị cho lần commit tiếp theo
-git add filename1 filename2 thêm một or nhiều file
-git add .: thêm toàn bộ tất cả file bị thay đổi

-git commit: Xác nhận và lưu trữ các file đã thay đổi
 + git commit -m "message commit"

 -git log: Xem lại lịch sử commit

 -Git status: Xem trạng thái hiện tại của thư mục làm việc, nó sẽ hiển thị những files thay đổi so với lần commit trước đó, đồng
 thời hiển thị trạng thái của các file đó như là đã được xác nhận để commit hay chưa

 -git push: đẩy tất cả commit lên trên remote repository
 -git clone: tải remote repository về máy tính

 -git checkout: chuyển sang một branch khác
  + git checkout branch-name: chuyển branch
  + git checkout -b branch-name: Tạo branch
